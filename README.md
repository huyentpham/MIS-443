## 📌 I. Introduction

In this project, we take on the role of **Data Analysts** for a B2B organization. The company uses a CRM system to track sales opportunities across regions, products, and industry sectors. However, the current raw data is fragmented, making it difficult for management to identify high-value assets and detect operational gaps.

The objective of this project is to build a **robust relational database**, clean the sales pipeline data, and use Python to visualize key metrics. This will provide strategic insights and support decision-making for the upcoming fiscal year.

---

## 🎯 II. Project Objectives

By completing this project, we have achieved the following objectives:

* Identify and evaluate an existing database-oriented data analysis project
* Understand database schema design and data flow in an analytical system
* Implement a database system and populate it with sample or real datasets
* Use SQL to perform data querying and management tasks
* Apply Python for basic data analysis and visualization
* Explain how database systems support business analytics processes
